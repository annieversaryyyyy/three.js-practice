###  Project Description

- **Road plane** where the car moves  
-  **3D car model (GLTF)** loaded via `GLTFLoader`  
-  **Lighting** ‚Äî `AmbientLight` and `DirectionalLight`  
-  **Camera** with `OrbitControls` for interactive view control
